CSS3 Gradients

CSS gradients are cool. You can do some pretty neat things with them :)

In the same way that the background of an element can be declared to be a certain color in CSS (e.g. background-color: #ecf0f1;), the background can be declared to be a gradient. Gradients can be done in two ways:

  • Linear
  • Radial
